Advanced Menu

Step 1: Select the range you want to fill

Step 2: List the colours you want, following the format in the example. Include the # sign and separate each colour with a comma then a space.

Step 3: List the rows you want each colour to start from, following the format in the example. Separate each row # with a comma then a space. Leave blank if you want the rows to be auto-calculated.

Step 4: Make sure the colours and rows match. You should have the same number of colours as rows.

Step 5: Click the checkbox if you want absolute row references.

Step 6: Click the Submit button!
